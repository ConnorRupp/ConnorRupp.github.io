Project Description

This backend application uses Node.js and Express to provide CRUD functionality for users and courses, along with integration with the OpenAI API. Users can upload course notes (PDFs), and the AI assistant generates multiple-choice questions based on the content. The system includes secure user registration and login, dynamic user ID handling, and storage with MongoDB.

Skills Learned

  • Building RESTful APIs with Node.js and Express
  • Integration with OpenAI API for AI-driven content generation
  • MongoDB for persistent data storage and dynamic user/course management
  • File uploads and server-side validation
